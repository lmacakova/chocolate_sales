## Chocolate Sales
Exploratory analysis of chocolate sales in 6 countries and prediction of chocolate sales for the next year with Random Forest Regressor model[^1].

## Data
Data came from [this dataset](https://www.kaggle.com/datasets/saidaminsaidaxmadov/chocolate-sales).
The dataset represents a global chocolate company tracking sales performance across products, countries, and employees. 

## How to work with chocolate_sales repository:
1. Clone the repository with:\
 bash
 ```
 git clone https://github.com/lmacakova/chocolate_sales.git
 cd chocolate_sales
 ```
2. Create a virtual environment:\
 bash
 ```
 python -m venv .venv
 ```
    # macOS/Linux
 ```
 source .venv/bin/activate
 ```
    # Windows PowerShell
 ```
 .venv\Scripts\Activate.ps1
 ```
3. Install requirements:\
 bash
 ```
 pip install --upgrade pip
 pip install -r requirements.txt
 ```
4. Open the notebook in VS Code or Visual Studio Code and select Python (applied) as the kernel.
